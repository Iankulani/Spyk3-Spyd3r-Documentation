// Step-by-Step

Installation Guide

Follow these steps to get Spyk3 Spyd3r running from the GitHub repository. Choose your method below.

01
Prerequisites
Ensure you have Git, Python 3.10+, and pip installed on your system.
# Check versions git --version python3 --version pip3 --version
02
Clone the Repository
Clone the Spyk3 Spyd3r repository from GitHub to your local machine.
# Clone via HTTPS git clone https://github.com/Iankulani/spyk3-spyd3r.git # Or via SSH (recommended) git clone git@github.com:Iankulani/spyk3-spyd3r.git # Navigate into the project cd spyk3-spyd3r
03
Create a Virtual Environment
Isolate dependencies using a Python virtual environment.
# Create venv python3 -m venv .venv # Activate on Linux / macOS source .venv/bin/activate # Activate on Windows .venv\Scripts\activate
04
Install Dependencies
Install all required packages from the requirements file.
# Install all dependencies pip install -r requirements.txt # For development extras pip install -r requirements-dev.txt
05
Configure Environment
Copy the example config and set your environment variables.
cp .env.example .env # Edit with your preferred editor nano .env # Required variables inside .env: BOT_TOKEN=your_bot_token_here TARGET_URL=https://example.com LOG_LEVEL=INFO
06
Run the Bot
Launch Spyk3 Spyd3r and watch it go to work.
# Run directly python3 spyk3-spyd3r.py # Run in background (Linux) nohup python3 spyk3-spyd3r.py > logs/output.log 2>&1 & # Check status tail -f logs/output.log
01
Import Core Modules
Import the Spyk3 Spyd3r package and its dependencies in your Python script.
import asyncio import os from dotenv import load_dotenv from spyk3_spyd3r import Spyd3rBot, Config from spyk3_spyd3r.modules import WebCrawler, Logger # Load environment variables load_dotenv()
02
Initialize Configuration
Set up the bot configuration with your desired settings.
# Build configuration object config = Config( token=os.getenv("BOT_TOKEN"), target_url=os.getenv("TARGET_URL"), max_depth=3, concurrency=5, log_level="INFO", output_dir="./results" )
03
Instantiate the Bot
Create a Spyd3rBot instance and attach modules.
# Create bot instance bot = Spyd3rBot(config) # Attach the web crawler module crawler = WebCrawler( depth=config.max_depth, follow_redirects=True, respect_robots=True ) bot.attach_module(crawler) # Set up logging logger = Logger(level=config.log_level) bot.set_logger(logger)
04
Define Hooks & Callbacks
Register event hooks to process data as the bot discovers it.
# Register on-page callback @bot.on("page_crawled") async def handle_page(page_data): print(f"Crawled: {page_data['url']}") await bot.store(page_data) # Register on-error callback @bot.on("error") async def handle_error(err): logger.error(f"Spider error: {err}")
05
Run the Async Event Loop
Start the bot using Python's asyncio event loop.
async def main(): await bot.start() await bot.run_campaign( target=config.target_url, duration=3600 # 1 hour ) await bot.stop() if __name__ == "__main__": asyncio.run(main())
06
Export & Analyse Results
Access collected data and export to your preferred format.
# After run completes, export results results = bot.get_results() # Export to JSON bot.export(results, fmt="json", path="./output.json") # Export to CSV bot.export(results, fmt="csv", path="./output.csv") # Print summary print(results.summary())
01
Pull the Docker Image
Pull the latest Spyk3 Spyd3r image from Docker Hub or build locally.
# Pull from Docker Hub docker pull spyk3spyd3r/bot:latest # Or build locally after cloning docker build -t spyk3-spyd3r .
02
Run the Container
Start the bot with your environment variables passed in.
docker run -d \ --name spyd3r \ -e BOT_TOKEN=your_token \ -e TARGET_URL=https://example.com \ -v $(pwd)/results:/app/results \ spyk3-spyd3r
03
Check Logs
Monitor the bot's output in real time.
docker logs -f spyd3r
